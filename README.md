# 💎 PRism

### AI 기반 동료평가로 완성하는 개인 브랜딩 및 팀 매칭 플랫폼
동료 평가를 통해 본인의 소프트 스킬을 AI로 분석하여 검증하고, 프로젝트를 게시하여 홍보 및 포트폴리오로 활용할 수 있는 무료 솔루션입니다.
